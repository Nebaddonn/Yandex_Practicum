Бизнес-контекст задачи

В аналитическое агентство обратилась компания, которая приобрела лицензию на проведение конференций TED. Сейчас идёт этап организации первого мероприятия: заказчик ищет выступающих, подбирает темы и площадки. 
Компании важно провести мероприятие в духе классических конференций TED и сделать это так, чтобы оно было запоминающимся и интересным. Для этого заказчик хочет учитывать опыт проведённых конференций и просит разработать дашборд, который в этом поможет.
